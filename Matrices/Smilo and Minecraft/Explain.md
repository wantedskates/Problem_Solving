# BFS Algorithm

This project demonstrates the Breadth First Search algorithm.

## Graph Structure

![Graph Diagram](images/WhatsApp Image 2026-05-15 at 20.07.44.png)

## Idea

BFS explores the graph level by level using a queue.

## Complexity

- Time: O(V + E)
- Space: O(V)

## Code Example

```cpp
queue<int> q;
q.push(start);
```
