# BFS Algorithm

This project demonstrates the Breadth First Search algorithm.

## Graph Structure

!![Graph Diagram](Matrices/smilo_minecraft/images/image.jpeg)

## Idea

BFS explores the graph level by level using a queue.

## Complexity

- Time: O(V + E)
- Space: O(V)

## Code Example

```cpp
queue<int> q;
q.push(start);
```
